Zusammenfassung

Schrift

  • Inter (rsms.me/inter)
  • Durchgestrichene Null via "zero" 1
  • Optimiert für UI und Screens
  • Gewichte: 400, 500, 600, 700

Farben

  • Primär: #00a86b + #e6960a
  • Status: Grün / Gelb / Rot
  • Dark/Light Mode mit Grünstich
  • Hohe Kontraste für Lesbarkeit

Cyberpunk

  • Corner Brackets auf Containern
  • Scanlines im Header/Footer
  • Glitch-Offset auf Headlines
  • Dezente Partikel im Hintergrund

Design-Prinzipien

  • Klarheit vor Dekoration
  • Status auf einen Blick (Farbbalken)
  • 8px Grid-System (Abstände)
  • Touch-freundlich (min. 44px)
